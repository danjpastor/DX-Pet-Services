# DX Pet Services

DX Pet Services is a World of Warcraft addon that expands battle pet collection, tracking, source information, and map functionality.

## Installation

1. Download the latest release.
2. Extract the `DX_PetServices` folder.
3. Place it in:

   `World of Warcraft/_retail_/Interface/AddOns/`

4. Restart World of Warcraft or reload the interface.

This project is under active development.
